## 功能说明
增加右手显示器显示Caps Lock、Num Lock、Scroll Lock状态功能
以“CNS”显示是否开启Caps Lock、Num Lock、Scroll Lock，如均为开启，则显示“---”，开启一项或几项，则显示“CNS”中对应的字母。
